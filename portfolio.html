Bike Alarm System

A fully functioning motion-sensing alarm system built for bicycle security. The alarm triggers automatically when the bike is moved, and can be silently disarmed using an RFID key fob — no app, no phone required.

02

Caseworker Communication Device

A purpose-built communication device designed to support social workers in the field. The device enables reliable messaging and delivers real-time updates about the individuals being assisted. A core feature is an integrated SOS system, allowing users to send emergency alerts directly to their caseworkers when it matters most.

03

Automatic Plant Watering System

An automated irrigation system built around a homemade capacitive moisture sensor. When soil dryness crosses a set threshold, the system triggers a relay to water the plant — no manual intervention needed. Fully self-contained and low power.
